# 노벨피아 다운로더

노벨피아에 업로드된 소설과 이미지를 마크다운 형태로 다운받습니다.

무료로 공개된 자유 소설만 다운로드 가능하며 로그인이 필요 없습니다.

## 면책 사항

이 프로그램은 노벨피아로부터 승인되지 않았으며 이용 약관을 위반할 가능성이 있습니다.

프로그램 사용으로 일어난 모든 문제는 사용자의 책임입니다.

다운로드한 파일은 타인에게 유포, 공유, 공개하면 안되며 다른 리더에서 읽는 등 개인적으로만 사용해야합니다.

## 사용법

### 1. 회차 목록 저장

`python3 main.py listup https://novelpia.com/novel/000000`에서 링크를 실제 링크로 바꾸고 실행하세요.

회자 정보와 설정이 담긴 `episodes_000000.txt` 형태의 파일이 생성되며 이 파일은 나중에도 재사용할수 있습니다.

### 2. 파일 수정

`episodes_000000.txt` 형태의 파일에서 다운받지 않으려는 회차를 삭제하고 설정을 원하는대로 조정하세요.

전체 설정은 아래에 나와있습니다. 기본 설정대로 다운로드하는것을 가장 추천합니다.

### 3. 다운로드

`python3 main.py download episodes_000000.txt`에서 파일명을 실제 파일명으로 바꾸고 실행하세요.

`output_<유닉스타임>/` 폴더 안에 글과 이미지(켠 경우)가 저장됩니다.

## 설정 값 (episodes_*.txt 상단)

| 설정 명 | 기본값 | 이게 뭔가요 |
|------|--------|------|
| `novel_no` | - | 내부적으로 고유번호를 알기 위한 값이며 수정하면 안됩니다 |
| `remove_free` | true | 제목 앞의 `무료` 표시를 제거합니다 |
| `download_images` | true | 이미지도 같이 다운로드 후 원래 글에 이미지가 있던 부분에 `(이미지-id)`로 치환합니다 |
| `order_prefix` | true | 파일명 앞에 `1_`, `2_` 같은 회차 번호 추가합니다 |
